# cMandel

![Example mandelbrot image](assets/images/example.png)

cMandel is a C program for generating and rendering mandelbrot sets.

## Dependencies

Install `libpng` using your system package manager.

## Building

```bash
# Builds
make all

# Cleans up
make clean

# Debug executable
make debug

# Runs
./cmandel
```
